## LabVIEW Demo  
**ChingMu VRPN for LabVIEW**

### 使用说明

1. 打开示例文件 **`CMVrpn_Demo_body.vi`**  
2. 输入服务器地址，选择刚体 ID  
3. 通过循环开关选择 **单帧取值** 或 **循环取值**  
4. 点击运行开始接收数据  

### 注意事项

- Demo 默认调用 **32 位 DLL**，需与 **LabVIEW 版本位数一致**  
- 如需使用 **64 位 DLL**，请在程序框图中手动修改 DLL 路径  
- 若需要接收 **多个刚体数据**，可参考 **`CMVrpn_Demo_2Body.vi`** 的程序框图结构进行扩展
